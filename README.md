# location

Spring boot 3.1.2
Java 17
JPA
H2

문서 : swagger (http://localhost:8080/swagger-ui/index.html)

# 과제 : 사용자와 위치정보(위경도)를 주고받는 백엔드 서버를 자유롭게 구성

## API
1. Member 등록
2. Game 등록
3. Member_Game 등록
4. 사용자 기준 위지 등록
5. 기준 위치로 부터 location 리스트 등록
6. 기준 위치로 부터 거리 조회

## 완성 못한 기능
websocket을 활용하여 실시간으로 기준 점으로 부터 위치 변경 이벤트가 발생할 때마다 거리 계산해주는 기능
